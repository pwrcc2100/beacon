ğŸ†˜ Troubleshooting

PDFs not generating?

Problem: Pandoc not installed Solution:

# Mac
brew install pandoc

# Linux
sudo apt-get install pandoc

Problem: LaTeX errors Solution: Install BasicTeX (smaller than full LaTeX):

# Mac
brew install --cask basictex

Presentation page not loading?

Problem: Material Icons not showing Solution: Icons are loaded via Google Fonts CDN in layout.tsx. Check your internet connection.

Problem: Styling looks broken Solution: Make sure Tailwind CSS is compiled:

npm run dev

Google Slides conversion issues?

Problem: Formatting looks wrong Solution: Use the manual slide structure in GOOGLE_SLIDES_SETUP.md rather than auto-conversion.
